## 主要动机
进一步提高整个应用的CPU利用率/交互性与任务之间的公平性

因此需要实现强制任务切换，并引入中断，可采用如下方法：

- 时钟中断：基于时间片进行调度
- （不在这里引入）串口中断：在发出输出请求后，不是轮询忙等，而是中断方式响应
